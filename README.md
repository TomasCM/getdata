Coursera. Getting and Cleaning Data
Course project . Version 0.0
==================================================================

This script summarized data from project about the accelerometers from the Samsung Galaxy S smartphone. A full project description is available at the site where the data was obtained:  http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones 

Source  data for the project: https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip  

The script select data with  "mean" or "std" in the name. Summarize data by subject and activity.
 

For each record it is provided:
======================================

- An identifier of the subject who carried out the experiment. 
- Its activity label. 
- A 86 data with mean of the variable as same name source. 



The dataset includes the following files:
=========================================

- 'README.txt', this document

- 'CodeBook.txt': List of all features.

- 'outcome.txt' Data sumary data

- run_analysis.R  Script
